I am an engineer and researcher with a computational ecology background from **UCL** and a biology background from **Sun Yat-sen University**, now pivoting hard into **algorithm engineering**, especially **search, recommendation, ranking, and backend systems**.

### Snapshot

- Building an end-to-end **Xiaohongshu-style search and recommendation system** with retrieval, ranking, recommendation, and backend service modules.
- Strong interest in **Algorithm Engineer** roles, especially in recommendation systems, search, ranking, data intelligence, and backend ML systems.

### Education
